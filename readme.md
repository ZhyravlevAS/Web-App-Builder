Gulp build web application
==========================

## Installation:

```bash
$ git clone https://github.com/ZhyravlevAS/Web-App-Builder.git
$ cd Web-App-Builder
$ npm install && bower install
```

## For run development server

```bash
$ git serve
```

## Run build of application

```bash
$ git build --env production
```

## On board
* Bootstrap SASS
* Font Awesome
* jQuery
* Modernizr
* Twig engine
